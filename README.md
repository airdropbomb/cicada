# cicada auto task

## Installation

1. Clone the repository:
```bash
git clone https://github.com/airdropbomb/cicada.git && cd cicada
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. input bearer & cookie in `accounts.txt` file in the root directory:
```bash
nano accounts.txt
```

privatekey2

4. ## Usage

1. Start the bot:
```bash
python3 bot.py
```
